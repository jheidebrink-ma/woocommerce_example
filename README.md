# WooCommerce example

1. Create a `.env` based on `.env_example`
2. Start docker
3. Go to your browser: http://localhost:82
4. Copy the folders from example_data/plugins to the public/wp-content/plugins folder.
5. Copy the folders from example_data/themes to the public/wp-content/themes folder.
6. Visit: http://localhost:82 and start configuring.
